September 2023 - Present

Software Engineering Intern

Google

This summer I had the opportunity to join Google as a Software Engineering Intern, and I couldn't have had a better time. My project was to write a dashboard application to centralize data about machine learning models for my team, App Ads. I was tasked with writing both the frontend and backend, based off of some existing code that had been written by my two hosts earlier in the year. I started by converting the existing backend structure from Golang into Python, adding significant functionality and eventually creating an HTTP server capable of automatically parsing data from thousands of internal files, storing that data in a globally distributed database, and serving the data through several API endpoints. I then created an Angular app to handle the frontend, which allowed users to view model data in two ways: a table, which listed the name of each model and linked to its internal location, and a D3.js tree, which displayed any models branched from the current production models and a wealth of data about each of them (including their most recent file changes, if any children were being live tested internally, and more). In just 12 weeks I was able to complete the entire dashboard and host it on an internal job scheduling framework so that it could run on its own, making this one of my quickest and most rewarding experiences yet.
